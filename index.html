Questions fréquentes

C'est quoi la clé API OpenRouter ? +
OpenRouter est un service gratuit qui vous donne accès à des modèles IA. Créez un compte sur openrouter.ai, générez une clé API gratuite et collez-la dans 87 Vision. Aucune carte bancaire requise.
La qualité des fiches est-elle bonne ? +
Oui ! L'IA analyse l'image en détail : marque, modèle, couleurs, matières, usage. Vous pouvez ensuite modifier n'importe quel champ directement dans l'interface avant de copier.
Mes données sont-elles sécurisées ? +
Votre clé API est stockée uniquement dans votre navigateur (jamais sur nos serveurs). Les photos sont envoyées directement à l'IA sans être stockées. Votre compte est protégé par Google OAuth et authentification 2FA.
Puis-je annuler mon abonnement ? +
Oui, à tout moment depuis votre espace client. Pas d'engagement, pas de frais cachés. Vous repassez sur le plan Free immédiatement.
